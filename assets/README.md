# Thư mục hình ảnh

Đặt ảnh theo nhóm để dễ thay vào giao diện:

- `images/banners/`: banner ngang, banner carousel, banner flash sale, banner AI.
- `images/products/`: ảnh laptop, màn hình, PC, phụ kiện và sản phẩm trong card.
- `images/categories/`: icon hoặc ảnh cho "Danh mục nổi bật".
- `images/floating-ads/`: banner dọc hai bên, nút quảng cáo nổi.
- `images/logos/`: logo Phong Vũ, logo hãng, nhãn chiến dịch.
- `images/icons/`: icon nhỏ dùng trong nút, badge, thông số sản phẩm.

Nên dùng định dạng `.webp` cho ảnh chính để nhẹ và mượt hơn. Nếu ảnh có nền trong suốt, dùng `.png`.
